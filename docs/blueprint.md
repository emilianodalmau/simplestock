# **App Name**: Firebase Blank Slate

## Core Features:

- Blank Page Display: Display a blank page with a clean and minimal UI.
- Firebase Integration: Connect the Next.js application to a new Firebase project for backend services.
- Authentication Setup: Implement basic user authentication (sign-up/sign-in) using Firebase Authentication.
- Firebase Firestore: Configure Firestore to store the content created using this blank slate

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to convey simplicity and calmness.
- Background color: Very light gray (#F5F5F5), near white, to keep the focus on the blank page content.
- Accent color: Muted teal (#80CBC4) for interactive elements to provide subtle contrast.
- Body and headline font: 'Inter', a grotesque sans-serif font known for its modern and neutral appearance.
- Use minimalist line icons for a clean and modern look.
- Keep the layout simple and centered for a focus on content creation.
- Subtle fade-in animations for page transitions and interactive elements.
